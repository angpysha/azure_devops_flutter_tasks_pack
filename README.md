# Azure DevOps Flutter Tasks Pack
[![Flutter](https://img.shields.io/badge/Flutter-blue?logo=flutter)](https://github.com/flutter/flutter)

Collection of Azure DevOps tasks for Flutter CI/CD pipelines.

## Documentation

- Full docs page: `docs/index.md`
- Marketplace details page source: `overview.md`

## What this extension helps with

- Build Flutter Android and iOS artifacts.
- Archive and re-sign iOS packages.
- Upload releases to Firebase, TestFlight, and Google Play.
- Handle utility steps such as env setup and build number updates.
